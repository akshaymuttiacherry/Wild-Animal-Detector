What if AI could help protect both wildlife and communities? 🦏

I built a real-time animal intrusion detection system that uses a custom YOLOv8 model to identify elephants, buffalo, rhinos, and zebras from video or webcam — and sounds an alert the moment one is detected.

The motivation is simple: human-wildlife conflict is a serious issue in many regions, and early detection can save lives on both sides.

Built using:
→ YOLOv8 for object detection
→ OpenCV for frame-by-frame video analysis
→ Pygame for real-time audio alerts
→ Python for the full pipeline

Small project, but a glimpse of how ML can go beyond benchmarks and into the real world.
